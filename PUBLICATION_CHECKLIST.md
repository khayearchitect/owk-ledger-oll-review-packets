# Publication Checklist

Use this checklist before creating or updating the public GitHub repository.

## Required

- [ ] Confirm all data is synthetic.
- [ ] Confirm no real client, employee, bank, tax, ID, registration, invoice, or payment data is included.
- [ ] Confirm no app source code is included.
- [ ] Confirm no credentials or environment files are included.
- [ ] Confirm no private planning notes are included.
- [ ] Confirm review wording does not request tax, legal, accounting, payroll, audit, or compliance advice.
- [ ] Confirm the boundary notice is visible in the README and packet.
- [ ] Confirm GitHub issue template asks for format feedback only.
- [ ] Confirm public-facing copy uses OWK's collective voice: `we` / `our`, not first-person singular.
