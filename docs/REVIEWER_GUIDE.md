# Reviewer Guide

## Purpose

This synthetic packet tests whether a structured business record pack can help a reviewer understand what is present, what is missing, and what needs clarification before formal review begins.

## Recommended Review Order

1. Read `packet/OLL_REVIEWER_READY_PACKET_BLUE_BASIN_SYNTHETIC.md`.
2. Check the Missing-Information Checklist.
3. Compare the CSV exports against the packet summary.
4. Review the source document summaries.
5. Record whether the packet would reduce or increase back-and-forth.

## Useful Feedback

Useful feedback identifies friction clearly:

- A required field is missing.
- A label is ambiguous.
- A section is useful but placed too late.
- A section creates false confidence.
- A CSV is useful but needs an additional column.
- A summary is helpful but needs a source reference.

## Boundary

The packet is not a professional report. It is a business-prepared review packet using synthetic data.
